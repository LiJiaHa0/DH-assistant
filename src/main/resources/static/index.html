📌 项目简介

DH Assistant 是一个面向企业客户的智能文档管理与问答系统。系统支持用户注册登录后，通过自然语言与 AI 助手进行对话，快速获取文档知识、解答业务问题，提升工作效率。

✨ 核心功能

  • 用户注册与登录（手机号 + 密码）
  • 智能问答对话（基于大语言模型）
  • 文档知识库管理
  • 对话历史记录
  • 多用户权限隔离

🛠 技术栈

Spring Boot 3.x MyBatis-Plus Sa-Token MySQL Redis Druid DashScope AI
